# CLIP 2 — Votre boutique, votre puissance
## COLObanes · Pub vendeurs · Prompt Sora

---

## FICHE TECHNIQUE

| Paramètre | Valeur |
|-----------|--------|
| **Durée cible** | 10 secondes |
| **Format** | 9:16 portrait (vertical) |
| **Résolution** | 1080×1920 minimum |
| **Ton** | Professionnel, fier, entrepreneurial — « c’est VOTRE marque » |
| **Voix** | Même voix que clip 1 (continuité), plus affirmée |
| **Musique** | Reprendre le beat clip 1 ou variation plus énergique |
| **Visuels source** | Dossier `visuels/` — 3 images |

---

## INSTRUCTIONS GLOBALES POUR SORA

1. Vidéo **10 s**, 3 plans (~3,3 s chacun), enchaînement **fluide** depuis le clip 1 (option : fondu entrant 0,2 s).
2. Montrer la **montée en puissance** du vendeur : catalogue → boutique perso → synthèse « un seul outil ».
3. Voix off : texte **exact**, ton **convaincant** — insister sur « votre », « à votre nom », « partout ».
4. Textes écran : courts, percutants, majuscules.
5. Fin du clip : **call-to-action implicite** vers l’inscription (clip 3).

---

## PLAN 1/3 — Catalogue pro (0:00 → 0:03,3)

### Référence visuelle
`visuels/03-catalogue-produits.png`

### Description visuelle pour Sora
Interface admin : ajout produit — photo, prix FCFA, stock, promotion. Animation : champs qui se remplissent rapidement, miniature produit qui apparaît dans une grille catalogue élégante. Sensation « 2 minutes et c’est en ligne ».

### Texte à l’écran
```
2 CLICS
CATALOGUE PRO
```

### Voix off — À DIRE MOT POUR MOT
> « Photo. Prix en francs CFA. Stock. Promotion. **Deux clics** — et votre catalogue est **beau**, **en ligne**, **prêt à vendre**. Vos clients n’attendent plus : **ils achètent**. »

### Animation
- Entrée : zoom sur formulaire produit
- Effet : checkmarks verts sur chaque champ validé
- Sortie : transition vers écran boutique

## PLAN 2/3 — Ma boutique · Partage (0:03,3 → 0:06,6)

### Référence visuelle
`visuels/04-boutique-partage-reseaux.png`

### Description visuelle pour Sora
Page boutique personnalisée vendeur + icônes partage WhatsApp, Facebook, Instagram, lien copiable. Animation : logo boutique pulse, icônes réseaux s’illuminent une par une, flèches « partager ». Fierté du vendeur.

### Texte à l’écran
```
MA BOUTIQUE
PARTAGEZ · VENDEZ
```

### Voix off — À DIRE MOT POUR MOT
> « **Votre** boutique. **Votre** nom. **Pas** le nôtre. Un lien — WhatsApp, Facebook, partout — et vos clients achètent **directement chez vous**. C’est ça, le vrai e-commerce. »

### Animation
- Icônes réseaux : apparition séquentielle 0,3 s chacune
- Lien boutique : effet « copié » avec flash vert

## PLAN 3/3 — Un seul outil (0:06,6 → 0:10)

### Référence visuelle
`visuels/05-synthese-un-seul-outil.png`

### Description visuelle pour Sora
Visuel synthèse : gestion + vente fusionnées, logo COLObanes central, flux commandes → catalogue → boutique. Énergie unifiée, bleu-orange, moderne. Sensation « tout est dans la même app ».

### Texte à l’écran
```
UN SEUL OUTIL
GESTION → VENTE
```

### Voix off — À DIRE MOT POUR MOT
> « De la gestion à la vente : **un seul outil**. **COLObanes** — le marché du Sénégal. Et la meilleure partie ? **Créer votre boutique, c’est gratuit.** »

### Animation
- Zoom central sur logo COLObanes
- Fin : texte « GRATUIT » apparaît en orange + fondu vers clip 3
