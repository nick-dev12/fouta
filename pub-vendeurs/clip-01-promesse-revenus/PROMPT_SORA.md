# CLIP 1 — La promesse · Revenus réels
## COLObanes · Pub vendeurs · Prompt Sora

---

## FICHE TECHNIQUE

| Paramètre | Valeur |
|-----------|--------|
| **Durée cible** | 10 secondes |
| **Format** | 9:16 portrait (vertical) |
| **Résolution** | 1080×1920 minimum |
| **Ton** | Inspirant, ambitieux, concret — parler au vendeur sénégalais |
| **Voix** | Homme ou femme, français **Sénégal**, chaleureux, persuasif, rythme dynamique |
| **Musique** | Beat moderne afro-pop léger, monte en intensité, pas de voix concurrente |
| **Visuels source** | Dossier `visuels/` — 3 images à enchaîner |

---

## INSTRUCTIONS GLOBALES POUR SORA

1. Générer **une seule vidéo continue de 10 s** en 3 séquences fluides (~3,3 s chacune).
2. Utiliser les **3 images de référence** comme base de chaque plan (morphing / Ken Burns / léger zoom — pas de coupe brutale).
3. **Voix off en français** : lire le texte **exactement** comme indiqué, sans ajouter de mots.
4. Afficher les **textes à l’écran** en gros, blanc ou orange COLObanes, police sans-serif moderne (Poppins).
5. Palette : bleu **#3564a6**, orange **#FF6B35**, fonds clairs ou cinématiques selon l’image.
6. Ne pas inventer d’autres marques. Finir le clip sur une **énergie montante** (transition vers clip 2).

## PLAN 1/3 — Bienvenue (0:00 → 0:03,3)

### Référence visuelle
`visuels/00-bienvenue-vendeur.png`

### Description visuelle pour Sora
Affiche premium 9:16 : vendeur africain confiant en boubou bleu, vortex bleu-orange de produits, smartphone montrant l’accueil COLObanes, skyline Dakar. Zoom lent sur le téléphone puis sur le visage du vendeur. Particules lumineuses, effet cinéma.

### Texte à l’écran
```
BIENVENUE
ICI, TOUT EST POSSIBLE
```

### Voix off — À DIRE MOT POUR MOT
> « Vous avez des produits. Vous méritez mieux qu’une simple vitrine. **Bienvenue sur COLObanes** — ici, **tout est possible** pour votre boutique. »

### Animation
- Entrée : fondu depuis le noir, 0,3 s
- Mouvement : zoom lent 100 % → 108 % sur le smartphone
- Sortie : fondu rapide vers plan 2
