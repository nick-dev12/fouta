Logos des marques de camions - COLObanes
=================================================

Ce dossier contient les logos des marques de véhicules poids lourds affichés dans le carrousel de la page d'accueil.

Fichiers actuels : placeholders SVG avec les initiales des marques.

Pour remplacer par les vrais logos officiels :
- mercedes.svg, scania.svg, volvo.svg, man.svg, daf.svg
- iveco.svg, renault-trucks.svg, isuzu.svg, hino.svg, tata.svg

Formats acceptés : .svg, .png, .jpg
Taille recommandée : 100x100 px minimum (cercle d'affichage)
